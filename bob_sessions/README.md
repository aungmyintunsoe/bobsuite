# Bob Sessions

This directory stores exported markdown reports from IBM Bob IDE tasks.

## Purpose

Use this folder to save:
- Code analysis reports generated by QA Sentry
- Documentation generated by Doc Engine
- Session logs and task results
- Demo outputs for judging

## Structure

```
bob_sessions/
├── README.md (this file)
├── analysis_reports/
│   └── (QA Sentry scan results)
├── documentation/
│   └── (Generated docs)
└── demos/
    └── (Demo session exports)
```

## Usage

When running BobSuite MCP through IBM Bob:
1. Export your session results as markdown
2. Save them here for documentation
3. Use for hackathon judging and demonstrations

## Note

This directory is for storing outputs and session records. It will be populated during actual usage of the MCP server.
